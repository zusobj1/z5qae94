How to use "Locker" App.

কিভাবে "Locker" অ্যাপ ব্যবহার করবেন।

  • What is a locker app..?
  • "Locker" এ্যাপ্স কি..?
  • Locker হলাে একটি অপেন-সাের্স সিকিউরিটি অ্যাপ, যদি কেউ আপনার স্ক্রিনে ব্যবহার করা পাসওয়ার্ড সঠিকভাবে খুলতে না পারে তাহলে এই অ্যাপটি ব্যবহারের মাধ্যমে পুরো ডিভাইস অটোমেটিক রিসেট নিবে।
  • অ্যাপটি মূলত আপনার ডিডাইস রিসেট হিসেবে কাজ করে। আপনি যদি চান আপনার ডিভাইসের কোনাে কিছু অন্য কাউকে জানতে দিবেন না, তাহলে এটি ব্যবহার করে আপনার ডিভাইস সুরক্ষা রাখতে পারেন। কেউ পাসওয়ার্ড দিয়ে চেষ্টা করলে ডিভাইস রিসেট নিবে ফলে অন্য কেউ আর আপনার ডিভাইসে থাকা কিছু দেখতে সক্ষম হবে না।
  •       ★ যেভাবে অ্যাপটি ব্যবহার করবেন।
  • প্রথমে অ্যাপ্সটি অপেন করুন‚ তারপর Admin Enabled এ ক্লিক করুন।
  • Activate this device admin app এ ক্লিক করুন।
  • আপনার ডিভাইসের লকটা কতবার পর্যন্ত ট্রাই করার সুযোগ থাকবে তা সিলেক্ট করুন। অতঃপর Enable এ ক্লিক করে দিন।
  • Yes এ ক্লিক করুন।
  • আলহামদুলিল্লাহ অ্যাপ্সটি এখন সঠিকভাবে আপনার ডিভাইসে সক্রিয় ভাবে কাজ করছে।
